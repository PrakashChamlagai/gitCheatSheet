Git Cheat Sheet

presented by TOWER Version control with Git - made easy

Create

Colone an existing repository
$ git clone ssh://user@domain.com/repo.git

Create a new local repository
$ git init

Local changes

Changed files in your working directory
$ git status

Changes to tracked files
$ git diff

Add all current changes to the next commit
$ git add

Add some change in <file>to the next commit
$ git add -p <file >

Commit all local changes in tracked files
$ commit -a

Commit previously staged changes
$ git commit

Change the last commit
Don't amend published commits
$ git commit --amend

Commit History

Show all commits, starting with newest
$ git log

Show changes over time for a specific file
$ git loc -p <file>

Who changed what and when in <file>
$ git blame <file>

Branches & Tags

List all exiting branches
$ git branch -av

Switch HEAD branch
$ git checkout <branch>

Create a new branch based
on your current HEAD
$ git branch<new-branch>

Delete a local branch
$ git branch -d <branch>

Mark the current commit with a tag
$ git tag <tag-name>

Update & Publishe

Lis all currelty congfigured remotes
$ git remote -v

Show information about a remote
$ git remote shoe <remote>

Add new remote repository, named<remote>
$ git remote add <shortname> <url>

Download all changes form <remote>,
but don't integrate into HEAD
$ git fetch <remote>

Download all changes and directly
merve/integrate into HEAD
$ git git pull <remote> <branch>

Publish local changes on a remote
$ git push <remote> <branch>

Delete a branch on the remote
$ git branch -dr <remote/branch>

Publish your tags
$ git push --tags

Merge & Rebase

Mege <branch> into your current HEAD
$ git merge <branch>

Rebase your current HEAD onto <branch>
Dont' rebase published commits
$ git rebase <branch>

abort a rebase
$ git rebase --abort

Continue a rebase after resolving conflicts
$ git rebase --continue

Use your configured merge tool to
solve conflicts
$ git mergetool

Use your editor tomanually solve conflicts and (after resolving Z) mark file as resolved
$ git add <resolved-file>
$ git rm <resolved-file>

Undo

Discard all local changes in your working directory
$ git reset --hard HEAD

Discard local changes in a specific file
$ git checkout HEAD <file>

Revert a commit (vy producing a new commit with contracry changes)
$ git revert <commit>

...and preserve all changes as unstaged changes
$ git reset <commit>

...and preserve uncommmitted local changes
$ git reset --keep <commit>
